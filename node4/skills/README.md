# Skills moved to expert packs

Pack-scoped methodology skills live under the shared catalog:

```text
experts/<packId>/skills/<skillId>/SKILL.md
```

Examples:

- `experts/pentest/skills/pentest-web-recon/`
- `experts/ctf/skills/ctf-web-recon/`

Node loads skills from the **installed pack copy** (or catalog default pentest).  
Do not add new skills here.
